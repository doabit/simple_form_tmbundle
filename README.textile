h1. SIMPLE_FORM.TMBUNDLE

A TextMate-bundle for the "Simple_form".

h2. What is Simple_form_?

"http://github.com/plataformatec/simple_form":http://github.com/plataformatec/simple_form

h2. Install this TextMate-bundle...

<pre>
mkdir -p ~/Library/Application\ Support/TextMate/Bundles
cd ~/Library/Application\ Support/TextMate/Bundles
git clone git://github.com/doabit/simple_form_tmbundle 'Simple Form.tmbundle'
osascript -e 'tell app "TextMate" to reload bundles'
</pre>
